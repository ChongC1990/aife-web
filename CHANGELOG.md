# CHANGELOG

## [1.0.0] - 2026-03-14

### Added
- 🎉 初始版本发布
- 基于 Gemini 3.1 Flash Lite 的轮辋螺栓压装曲线 AI 视觉检测系统
- Web 界面支持拖拽上传图片
- 自动判断压装工艺合格性（合格/压力过高/未压到位）
- 本地历史记录功能（最近 10 条）
- JSONL 格式日志记录
- 图片自动压缩（macOS sips / Linux imagemagick）

### Technical Details
- **前端**: 原生 HTML/CSS/JS（零依赖）
- **后端**: Node.js 原生 HTTP Server
- **AI 模型**: gemini-3.1-flash-lite-preview (via SkillFree API)
- **端口**: 3456
- **部署**: Ubuntu 24.04 LTS (221.237.11.62)

### Validation
- 经 8 个真实工厂样本验证，准确率 100%
- 样本来源：2026-03-09 工厂现场
  - 4 张合格品
  - 76 张不合格-压力过高
  - 48 张不合格-压力过高且未压到位

---

## 判断逻辑

| 条件 | 结论 |
|------|------|
| 复位按钮左侧数值 = `10000` 且行程 ≥ 10mm | ❌ 压力过高 |
| 复位按钮左侧数值 = `10000` 且行程 < 10mm | ❌ 压力过高且未压到位 |
| 数值 ≠ `10000` 且行程 ≤ 9mm | ❌ 压力过高且未压到位 |
| 其余 | ✅ 合格 |
